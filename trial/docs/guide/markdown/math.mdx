# 数学公式
> Predoc 使用 `rehype-katex` 和 `remark-math` 进行公式渲染

KaTeX 是一个高效的 JavaScript 库，用于在网页上渲染数学表达式，支持 LaTeX 语法。
基础语法包括加减乘除、指数和根号、括号、运算符、文本和间距等常用数学符号的表示方法。
进阶部分涵盖求和与积分、矢量与矩阵、希腊字母、箭头与逻辑符号、其他常用符号以及定义和引用等更复杂的数学表达方式。
该工具适用于需要在网页中嵌入复杂数学公式的应用场景。

## 单行公式

> 使用 单个 `$` 包裹起来的是单行公式

The circumference of a circle is $2\pi r$.

## 多行公式

> 使用两个 `$$` 包裹起来的是单行公式

$$
\left\{
\begin{aligned}
x^2 & , \quad x > 0 \\
0 & , \quad x = 0 \\
-x & , \quad x < 0
\end{aligned}
\right.
$$

## 示例

```md
Lift($$L$$) can be determined by Lift Coefficient ($$C_L$$) like the following
equation.

$$
% \f is defined as #1f(#2) using the macro
\f\relax{x} = \int_{-\infty}^\infty
    \f\hat\xi\,e^{2 \pi i \xi x}
    \,d\xi
$$
```

渲染结果：

Lift($$L$$) can be determined by Lift Coefficient ($$C_L$$) like the following
equation.

$$
\f\relax{x} = \int_{-\infty}^\infty
    \f\hat\xi\,e^{2 \pi i \xi x}
    \,d\xi
$$

:::success
注意：该公式需要自定义宏：`math: {"\\f": "#1f(#2)"}`
:::
